# TCGA Kidney Cancers RNA Sequence Analysis

Dataset: https://portal.gdc.cancer.gov/projects/TCGA-KICH  

This dataset contains high-dimensional bulk RNA-Seq transcription profiles used to classify individuals into three specific kidney cancer subtypes (KICH, KIRP, and KIRC).  
The project implements a machine learning pipeline—including PCA for dimensionality reduction and a comparison between Logistic Regression and Random Forest models—to predict cancer pathology from genomic signatures.
